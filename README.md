# Research Papers
By Joseph Juma

## About
This is an archive aggregating my independent research papers. This is a collection of miscellaneous papers created in the course of my work which I'm unconcerned with publishing in any formal journal.

Papers are organized by the month and year of when they were written. Both the rendered `*.pdf` and corresponding `*.tex` and `*.bib` files are provided for papers created with LaTeX. For papers produced in other software, a `*.pdf` is provided with (optionally) an accompanying markdown (i.e. `*.md`) file.
